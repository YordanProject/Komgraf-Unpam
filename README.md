Komputer Grafik
Komputer Grafik adalah cabang ilmu komputer yang mempelajari cara membuat, memanipulasi, dan menampilkan gambar atau visual menggunakan komputer. Bidang ini menjadi fondasi di balik tampilan layar yang kita lihat sehari-hari, mulai dari antarmuka aplikasi, film animasi, hingga permainan video.

Pada Pembelajaran di semester 2 Universitas Pamulang, dalam mata kuliah Komputer Grafik. Kami diarahkan untuk menggunakan JupyterLab sebagai media pembelajaran Komputer Grafik.

Berikut adalah Library yang digunakan:
- Matplotlib
- Numpy
- Pygame
- CV2
